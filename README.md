
# 线上书城前端项目
### `概述`
这是sjtu-se web课程书城前端代码，css设计参考是扒的一个牙科医院的前端页面，所以应该不会有雷同的前端页面。对于这门课的后来者，我的**建议**是善用老师的demo！自己写会比较累~~~

### `项目运行`

终端： npm start 即可，也可以配置一下，用绿色的三角按钮运行
